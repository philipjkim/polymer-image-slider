##&lt;polymer-image-slider&gt;

`polymer-image-slider` allows user to show multiple images one at a time, with `prev` and `next` buttons.

Example:

```html
<image-slider images='["images/a.jpg","images/b.jpg","images/c.jpg"]'></image-slider>
```
